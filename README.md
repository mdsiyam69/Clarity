## 📦 What is Clarity?
Clarity is designed to make your financial analysis easier. It uses a unique architecture called Claude-skill and follows a Planning-with-Files approach. This enables it to coordinate different specialized sub-agents effectively. It keeps track of task plans, research findings, and logs to help you handle complex financial workflows with confidence.

Once you have installed Clarity, you can start using it to simplify your financial analysis:
